Sounds
Plays a sound when someone takes warden.
plays a sound when someone leaves warden(die, killed, admin and when they leave them self).

Marker
Push f Key to place a marker on the ground where you are aimed at.

Commands
sm_w / sm_warden - Allows the player taking the charge over prisoners.
sm_uw / sm_unwarden - Allows the player to retire from the position.
sm_rw - Allows admins to remove the current warden if available.
Inspect key (f) - place a warden marker where you are looking.
